Studio Olafur Eliasson

The body.home seems to contextualize that this page is the home page. I confirmed this by going to a different page and the class changed to body.archive-artwork

The .compass-wrapper seems to be containing the floating iceberg thing. Diving deeper into the DOM, it seems to just be a looping .mp4 video that acts like a link when clicked. It's a very interesting use of a "floating" element.

The header seems to house two buttons: one for "Olaffur Eliasson" that links to the index, and "Menu" which opens up a menu drawer to the right. This button must be what controls the aside.right-off-canvas-menu

There is an interesting behavior that happens when the menu is open... it activates a gradient on .exit-off-canvas that provides a psychedelic atmosphere when the main content is not in focus.

The most interesting DOM was the div.post-item. This seemed to be a repeated "component" within the container .parallax-content. The structure of the markup was the same, but the content was different. As I paginated through the blog posts, I also realized there there were multiple "posts types". A few I encountered were .type-image, .type-text, and .type-video. There were slight difference between the HTML markup for the different post types. I think this was to accomodate different content for the different post types.
